CSS

CSS position

  • static - (default)
  • relative
  • absolute - (Used for navbar/sidebar/overlay/popup)
  • fixed - (Used to center an element)
  • sticky
  • ("absolute" & "fixed" has no own spot)
  • ("absolute" can push the element outside of parent)
  • ("fixed" doesnt care about the parent, it is always relative to the viewport)

CSS units

  • px (pixels)
  • % (percentage)
  • em (relative to font-size of the element)
  • rem (relative to font-size of the root element)
  • vh (viewport height)
  • vw (viewport width)
  • vmin (viewport minimum)
  • vmax (viewport maximum)
